1. BIS 운영단말

BIS 운영단말은 우리나라 각 지자체의 버스 상태 및 BIT(버스안내기)를 운영관리하는 프로그램입니다.

저는 천안, 고양, 하남, 세종, 목포시 교통정보센터에 BIS(버스 운영 관리 시스템) 운영단말 웹 프로젝트 참여했습니다.

설계단계에서 해당 지자체에 직접 방문하여 서버와 디비의 스펙 상태를 확인 후 개발, 설치, 감리까지 받았습니다.

네이버, 카카오 지도 API와 Toast Grid를 이용해 버스 운영에 필요 한 기본적인 정보를 등록, 수정, 표출 하고

FTP 서버에 BIT(버스안내기)에 필요한 기반정보를 올려주었으며 BIT 장비 제어를 위해 프로토콜 문서 기준으로 TCP나 UDP 통신으로 장비 제어를 했습니다.

배포환경 : Windows10, Linux





2. ITS 운영단말

ITS 운영단말은 우리나라 각 지자체의 교통상황을 운영관리하는 프로그램입니다.

관리에는 VDS, DSRC, CCTV, VMS 등 도로에 설치되어있는 장비와 서버가 데이터를 주고 받아 그 데이터를 토대로 운영합니다.

저는 인천, 고양, 파주 ITS 운영단말 웹 프로젝트에 참여했습니다.

DSRC/RES, AVI, 통계페이지 다수 와 사용자 관리 페이지를 맡아 개발하였습니다.

네이버, 카카오 지도 API와 Toast Grid를 이용해 데이터를 표출했고 High Chart를 이용해 통계 그래프를 표출했습니다.

배포환경 : Windows Server 2008, Windows10, Linux

1) 인천 교통정보

2) 고양 교통정보

직접 개발은 통계자료 표출 화면을 했으며, 각 지자체에 웹 서버(Apache)와 WAS 서버(Tomcat)를 설치하고 연동 후 SSL 등록을 주로 하였습니다.

배포 환경 : Windows Server 2008, Linux

개발 환경 : 전자정부프레임워크 3.6, Java 1.8, Gradle, JPA, Tibero
